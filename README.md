```
⚡ Felix Kantin - Premium POS System
Sistem Kasir (Point of Sales) modern yang dirancang khusus untuk manajemen kantin. Memiliki antarmuka yang bersih, responsif, dan fitur analisis data real-time.
```
-----------------------------------------------------------
```
✨ Fitur Unggulan
- Interactive POS: Antarmuka kasir yang cepat dengan sistem keranjang belanja berbasis JavaScript.
- Real-time Analytics: Laporan pendapatan harian, mingguan, hingga bulanan.
- Product Management: Kelola stok, kategori, dan harga menu dengan mudah.
- Transaction History: Riwayat transaksi lengkap dengan rincian item yang dibeli.
- Modern UI/UX: Menggunakan Tailwind CSS dengan font Plus Jakarta Sans untuk tampilan premium.
```
-----------------------------------------------------------
```
🛠️ Tech Stack
- Backend: Laravel 10 (PHP 8.1+)
- Frontend: Tailwind CSS, Alpine.js, FontAwesome.
- Database: MySQL.
- Tools: Vite.
```
-----------------------------------------------------------
```
🚀 Instalasi Lokal

1. Clone Repository
   git clone [[https://github.com/username-kamu/pos-kantin-felix.git](https://github.com/username-kamu/pos-kantin-felix.git)](https://github.com/J-CNQU/Laravel-Cafe-POS-System.git)
   cd pos-kantin-felix

2. Instal Dependencies
   composer install
   npm install && npm run dev

3. Konfigurasi Environment
   - Salin .env.example menjadi .env
   - Buat database baru di MySQL bernama pos_kantin
   - Jalankan perintah: php artisan key:generate

4. Migrate & Seed
   php artisan migrate --seed

5. Jalankan Server
   php artisan serve
```

🛠️ Langkah Push ke GitHub via Terminal

Buka terminal di dalam folder proyek kamu dan jalankan perintah berikut secara berurutan:

1. Inisialisasi Git
   git init

2. Tambahkan Semua File
   git add .

3. Commit Pertama
   git commit -m "feat: initial commit POS system with analytics and fixed history bug"

4. Buat Repository di GitHub
   - Buka GitHub > New Repository.
   - Beri nama pos-kantin-felix.
   - Klik Create Repository.

5. Hubungkan dan Push
   (Ganti URL di bawah dengan URL dari repositori kamu)
   git remote add origin [[https://github.com/username-kamu/pos-kantin-felix.git](https://github.com/username-kamu/pos-kantin-felix.git)](https://github.com/J-CNQU/Laravel-Cafe-POS-System.git)
   git branch -M main
   git push -u origin main
